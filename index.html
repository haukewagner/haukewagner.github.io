Numpy notes

1. 创建array的函数

a = np.array([[1,2,3],[4,5,6]]) 必须是矩阵，如果其中有一个是字符串，其它自动转换成字符串

2. 特殊array构造

a = np.zeros(2)

b = np.zeros((2,3))
全零array，数据为浮点型 c = np.ones(3) 全一array，数据为浮点型 d = np.empty(4) 空array，内存的随机值 e = np.arange(5) 0-4的值 f = np.arange(1,9,2) [1 3 5 7],初值，末值(不包含)，步长 g = np.linspace(0, 10, num = 5) [ 0. 2.5 5. 7.5 10. ]，创建区间内的值，初值，末值(包含)，值的个数 h = np.ones(2, dtype = np.int64) 设置dtype, 默认为np.float64

3. 排序和合并函数

a = np.array([2,1,3,9,5,8]) b = np.sort(a) 排序函数，默认为升序 np.array([[2,1,3],[9,5,8]]) 会分别排序 c = np.array([[2,1,13], [9,5,8]]) d = np.concatenate((c[0], c[1])) 可以合并array，但参数为字典形式 相关其他函数argsort,lexsort,searchsorted,partition

4.查询array的大小，形状

a = np.array([[[2,1],[9,5],[4,7]],[[9,6],[2,1],[7,3]],[[1,2],[3,4],[5,6]]]) print(a.ndim) 3，a的维度 print(a.size) 18, 个数 print(a.shape) (3, 3, 2)
